# promo

Public gallery of brand films and interactive demos, served via GitHub Pages.

**Live:** https://sunprema.github.io/promo/

## Structure

```
index.html        landing page (gallery of all projects)
kitgig/index.html  Kitgig brand film
```

## Adding a project

1. Create a new folder with an `index.html` (and any assets it needs).
2. Add a card to the root `index.html` linking to the folder.
3. Commit and push to `main` — Pages redeploys automatically.

Each project lives at `https://sunprema.github.io/promo/<folder>/`.
